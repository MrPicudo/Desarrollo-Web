Etiquetas de texto en HTML:

Las etiquetas de texto nos permiten seleccionar un fragmento de texto y dotarlo de un significado especial, por ejemplo:

  • Se usa (p) para crear un párrafo
  • Se usa (br) para agregar un salto de línea
  • Se usa (pre) para mostrar texto tal cual se escribió en el editor
  • Texto importante o palabras clave (strong)
  • Texto en negritas pero sin importancia especial (b)
  • Texto con énfasis (em)
  • Texto en cursiva pero sin intención de énfasis (i)
  • Texto con subrayado (u)
  • Texto tachado con línea media (s)
  • Texto resaltado con rotulador amarillo (mark)
  • Texto normal con anotaciones pequeñas (small)
  • Números con exponentes 52=25 (sup)
  • Números con subíndices x1= 5, x2 = 8 (sub)
  • Para hacer una cita de otro sitio web usamos (q)

Dentro de un párrafo podemos usar los atributos de aling:

  • =left Para alinear el texto a la izquierda.
  • =right Para alinear el texto a la derecha.
  • =center Para centrar el texto.
  • =justify Para justificar el texto.
